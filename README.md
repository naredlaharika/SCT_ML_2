# Customer Segmentation using K-Means Clustering

This project focuses on segmenting customers of a retail store based on their purchasing behavior using the K-Means Clustering algorithm, an unsupervised machine learning technique.
The model analyzes features such as Annual Income and Spending Score to group customers into distinct clusters. This helps in understanding customer patterns and supports targeted marketing strategies.

## Objectives
- Perform customer segmentation using clustering techniques
- Identify optimal number of clusters using the Elbow Method
- Visualize customer groups for better insights

## Tools & Technologies
- Python
- Google Colab
- Pandas
- Matplotlib
- Scikit-learn

## Dataset
Mall Customer Dataset

## Outcome
Customers are grouped into meaningful clusters that help businesses improve marketing strategies and customer targeting.
